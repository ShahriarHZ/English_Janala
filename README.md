# 🌐 English Janala (ইংরেজি জানালা)

**English Janala** is a simple and responsive website to help users learn English vocabulary through interactive lessons.

## ✨ Key Features
* 📚 **Lessons:** Click a lesson to load words instantly from an API.
* 🔍 **Search:** Find any word across all lessons quickly.
* 🔊 **Pronounce:** Click the speaker icon to hear the word out loud.
* 📱 **Responsive:** Works perfectly on mobile, tablet, and desktop.
* ❓ **FAQ:** A clean accordion section for common questions.

## 🛠️ Built With
* **HTML5** & **JavaScript (ES6+)**
* **Tailwind CSS** (Styling)
* **DaisyUI** (UI Components)
* **Programming Hero API** (Data Source)

## 🚀 How to Run
1. Download or Clone this repository.
2. Open `index.html` in any web browser.
3. Start learning!
